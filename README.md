# Mathe Quest Hamburg (8. Klasse)

Spielerisches Mathe-Lerntool fuer Gymnasium Hamburg (Klasse 8).

## Enthaltene Features

- 3 Spielmodi: `Challenge`, `Zeitjagd`, `Bosskampf`
- 5 Themen: Brueche, Prozent, Gleichungen, Geometrie, Funktionen
- Punkte, XP, Level, Muenzen, Lernserie
- Hamburg-Missionskarte mit Sternen je Thema
- Norden-Fokus bei Stadtteilen: Wellingsbuettel, Poppenbuettel, Sasel, Bramfeld, Farmsen-Berne
- Tages-Challenge mit Bonus
- Badges fuer Motivation
- Alle Fortschritte werden lokal im Browser gespeichert

## Lokal starten

```bash
cd "/Users/niclas/Documents/New project 3/mathe-quest-hamburg"
python3 -m http.server 8773
```

Dann im Browser oeffnen:

`http://127.0.0.1:8773`
